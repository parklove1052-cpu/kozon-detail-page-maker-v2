# 레퍼런스(프리랜서) 저장소

상세페이지 제작자 도구가 사용하는 **레퍼런스(프리랜서) 명세 모음**입니다.

UI의 "레퍼런스" 드롭다운에서 선택하면, 해당 작가의 디자인 패턴·카피톤·이미지 디렉션이 자동으로 적용됩니다.

## 폴더 구조

```
references/
├─ README.md             ← 이 파일
├─ freelancers/          ← 프리랜서 1명당 1개 JSON
│   ├─ kwayeon.json      ← 과연 (크몽 베스트셀러)
│   └─ (추후 추가)
```

서버는 `freelancers/*.json` 을 자동 스캔해서 `/api/references` 로 노출합니다. 파일을 새로 추가하면 서버 재시작 없이 즉시 반영됩니다 (config 핫리로드 동일 패턴).

## JSON 스키마 (v1)

| 필드 | 필수 | 설명 |
|---|---|---|
| `key` | ✅ | URL/내부키. 영문 슬러그. 예: `kwayeon` |
| `label` | ✅ | UI 드롭다운 표시명 |
| `platform` | | "크몽" / "라우드소싱" 등 |
| `url` | | 작가 프로필 URL |
| `tagline` | | 작가의 한 줄 슬로건 |
| `source_notes` | | 분석 근거 메모 (별점/리뷰수/대표 작품 등) |
| `design_tokens` | ✅ | 컬러/폰트/사이즈/스페이싱/모서리 정의 |
| `tone` | ✅ | 형용사 배열 (감성적/모던/캐주얼 등) |
| `copy_lexicon` | | 자주 쓰는 단어 — Claude 카피 생성 시 힌트 |
| `copy_dont` | | 절대 쓰지 말아야 할 패턴 |
| `image_direction` | ✅ | 이미지 톤 (조명/팔레트/배경/무드) |
| `image_prompt_overrides` | | 영문 — ChatGPT 이미지 프롬프트에 그대로 주입 |
| `section_pattern` | ✅ | 섹션 흐름 패턴 (이 사람만의 구성 순서) |
| `do_dont` | | 카피·디자인 원칙 |

## 새 프리랜서 추가 절차

1. 사장님이 마음에 드는 프리랜서 URL 공유
2. `benchmark-analyzer` 에이전트로 분석 (또는 수동 분석)
3. 위 스키마대로 `freelancers/<slug>.json` 작성
4. 서버 재시작 없이 UI 새로고침 → 드롭다운에 자동 노출

## 현재 등록된 레퍼런스

| 키 | 라벨 | 결 |
|---|---|---|
| `kwayeon` | 과연 (크몽 베스트셀러) | 감성·섬세·프리미엄, 그레이/골드, 명품 어휘 |

(없음 = 레퍼런스 적용 안 함, 디자인 스타일만으로 생성)

## 사장님이 직접 추가 요청 시 양식

```
[레퍼런스 추가 요청]
URL: ...
선호 이유: ...
어떤 상품군에 쓸지: ...
```

위 양식으로 주시면 분석 후 JSON 작성합니다.
