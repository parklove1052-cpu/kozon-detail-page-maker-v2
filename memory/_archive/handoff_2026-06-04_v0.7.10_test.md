## 사장님 테스트 시나리오 (포스닉 에너지바 v.2)

### Step 1
- 제품 사진 첨부 (사장님이 첨부하신 ChatGPT Image PNG들)
- 「상세페이지 내용」 칸에 **포스닉 에너지바 기획안** 붙여넣기 (v.2 효능 표현 제거판)
- 디자인 스타일 = **모던**, 내용 스타일 = **정보형**, 레퍼런스 = **없음**

### Step 2 ① 기획 + 이미지 프롬프트 생성
- 응답 후 화면 하단 우측 build 식별자 `v0.7.9` 보이는지
- 슬롯 카드 그리드에 청사과/복숭아/민트/레드불 향별 슬롯 + Hero/Spec/라이프스타일 등 보임
- 슬롯 카드 ✓/✗ 토글 동작 + 모델·비율 셀렉트 정상
- 좌측 dropzone에 추가 이미지 drag → 정상 떨어짐

### Step 3 🎨 Higgsfield 일괄 생성
- `permission denied` 에러 없이 진행 (v0.7.10 패치 효과)
- 결과 PNG가 사장님 제품 그대로 + 배경만 변경된 형태 (used_attached:true 보강)
- 실패 슬롯은 🔄 재생성 / ✏️ Inpaint 동작

### Step 4 ② 상세페이지 HTML 생성
- 자동 생성된 PNG가 state.slotImages에 매핑되어 HTML에 그대로 사용됨
- output/ 폴더에 저장된 detail_*.html + JPEG zip 다운로드 정상
