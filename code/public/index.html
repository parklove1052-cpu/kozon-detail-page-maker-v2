📦 상세페이지 만들기

1 기본 입력 (카피·스타일·레퍼런스·참고 이미지)
제품 제품 사진 첨부 이 제품을 그대로 사용 · 배경/씬만 변경 요청

제품 사진을 여기로 끌어다 놓거나 클릭

제품을 기준으로 배경·씬을 바꾸는 프롬프트가 생성됩니다

기타 기타 이미지 첨부 상세페이지에 그대로 활용할 이미지 (선택)

활용할 이미지를 여기로 끌어다 놓거나 클릭

비워두면 OK — 부족한 이미지만 자동으로 프롬프트가 생성됩니다

이미지가 아직 없으면 — 영문 프롬프트를 받아 ChatGPT로 이미지 만든 뒤 Step 3에 채워 진행

이미지가 이미 다 있으면 — 기획·HTML 생성을 자동 연쇄로 한 번에 완성 (Step 2·3 자동 처리)

2 이미지 프롬프트 + ChatGPT로 만들기 (슬롯별 영문 프롬프트 확인 → ChatGPT에서 이미지 생성)
슬롯 0개 · ChatGPT에서 만든 이미지를 Step 3 슬롯에 업로드해 주세요

3 ChatGPT 결과 이미지 업로드 (ChatGPT에서 만든 PNG를 슬롯별로 끌어다 놓거나 클릭)

Step 2 영문 프롬프트로 ChatGPT에서 만든 이미지를 아래 슬롯에 채워주세요. 슬롯이 다 채워지면 Step 4 ② 버튼이 활성화됩니다.

4 최종 생성 (기획 + 이미지 → 단일 HTML)
